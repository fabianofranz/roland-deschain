Somewhere, someone is seeing something really nice right now. What if you could see through people's eyes? Like Roland Deschain, now you can.
Tech info: sample app running on OpenShift (PaaS by Red Hat). Java EE 6 via JBoss AS 7, HTML5 and CSS3 (Boilerplate), Instagram API.
